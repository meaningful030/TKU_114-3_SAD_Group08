# 0713 需求工程與程式碼代理成果

## 內容

- `interview_guide.md`：三類角色的非誘導式訪談大綱。
- `interview_notes.md`：清楚標記為模擬的訪談紀錄。
- `requirements_source_table.md`：13 筆可追溯來源與納入判斷。
- `candidate_requirements.md`：功能、品質、限制與業務規則。
- `code_agent_implementation_brief.md`：第一切片的小型規格書。
- `code_agent_log.md`：提示詞、採用／不採用與人工修正紀錄。
- `acceptance_report.md`：需求與畫面驗收結果。
- `first_working_slice/`：可直接開啟的純前端原型。

## 操作方式

1. 開啟 `first_working_slice/index.html`。
2. 保持「攤主」角色，故意留空欄位送出，確認出現錯誤提示。
3. 填入攤位名稱、作品類型與聯絡 email 後送出。
4. 確認畫面顯示申請編號與「審核中」。
5. 點選「主辦方」角色，確認待審清單出現同一筆資料。

原型資料只存在目前頁面記憶體，重新整理後會回到初始假資料，不代表正式送件或保存。
