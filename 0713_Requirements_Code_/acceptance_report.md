## 已知限制

- 重新整理頁面會清除申請，這是刻意遵守「無正式資料庫」的限制。
- 審核、排位衝突與地圖搜尋屬後續切片，本次只有待審清單入口。
- 模擬訪談尚不能代替真實主辦方與攤主證據。
- 產生的申請編號只供同一頁面展示，不保證跨工作階段唯一。
