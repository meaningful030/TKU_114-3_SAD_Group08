## 使用規則

1. 沒有填入「小組決定」的項目，不得從 `【待小組決定】` 改成「已確認」。
2. 若決定與目前原型不同，先更新需求、驗收條件與優先表，再修改程式。
3. 每個決定要保留來源：小組決議、真實訪談、正式簡章或教師指示。
4. Codex 可以依你填好的決策修改文件與程式，但不能替你選答案。
