# 使用者故事清單

## 文件說明

- 專案：第 8 組－同人誌販售會攤位管理系統「攤集」。
- 小組已決定採用下列 8 則使用者故事（決策 D-11）。
- 模擬訪談是課堂角色扮演，來源強度標為「中」；章程與可行性分析是既有小組文件，來源強度標為「強」。
- `AI-IDEA-01、AI-IDEA-02` 未作為任何使用者故事的需求來源。
