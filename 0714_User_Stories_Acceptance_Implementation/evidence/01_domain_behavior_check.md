# 證據 01：領域行為檢查

## 執行方式

使用工作區內建 Node.js 執行：

```text
node --check domain.js
node --check app.js
node domain.test.js
```

## 實際結果

```text
domain.test.js: 14 項行為檢查通過
exit code: 0
```

## 覆蓋內容

- 有效攤名、類型與 email 通過驗證。
- 空白攤名、空白類型與 `abc@` 同時被拒絕。
- 空白審核意見不得退回，狀態維持審核中。
- 有效退回意見去除前後空白後完整保存。
- 通過後狀態正確；待審計數只計審核中。
- 攤主狀態投影包含結果與意見，但不包含 email。
- 選擇「其他」但未補充類型時被拒絕。
- 退回後修改原申請可保留同一編號、回到審核中、清除目前退回提示並增加重送次數。
- 「其他：攝影集」可正確顯示。
