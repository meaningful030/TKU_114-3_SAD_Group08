## 訪談後人工判斷

1. 本次第一切片選「攤主送出申請，主辦方待審清單立即出現」；它最短、可操作，且串起兩個主要角色。
2. 搜尋、排位與衝突警告保留為候選需求，不在本次切片實作，避免擴張。
3. 「補件流程、禁排格、多格攤位、作者別名搜尋」證據不足，需正式訪談後才能納入。
