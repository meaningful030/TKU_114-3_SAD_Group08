## 2. 產出摘要

本次修正既有第二切片，沒有建立第三切片：

1. 在領域層建立互斥的待審與已審查詢投影。
2. 主辦方工作台拆成「待審案件」與「已審結果」兩區。
3. 通過或有效退回後，案件離開待審區但不刪除原資料。
4. 已審案件可由獨立入口唯讀回查，不能再次通過或退回。
5. 空白或全空白退回意見仍維持審核中及待審位置。
6. 退回案件補正重送後，沿用同一編號回到待審區；已審活動列移除。
7. 新增 5 組領域分流回歸測試及瀏覽器端驗收腳本。
8. 保留三張可閱讀的繁體中文畫面證據。
